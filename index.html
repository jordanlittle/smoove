Mobile grid units use the syntax "unit-mobile-X" with X being fourth, third, half, two-thirds, three-fourths. The normal unit-X class applies on larger screens, but display full-width and with no floats on smaller screens. On those smaller screens, the unit-mobile classes will apply. If you plan on using "unit-" classes after a set of "unit-mobile-" classes, use <div class="cf--mobile"></div> to clear the floats.
