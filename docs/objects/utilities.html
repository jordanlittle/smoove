Utilities

These are small helper classes that usually only do one thing.

Add a class of text-left for left-aligned text.

Add a class of text-center for centered text.

Add a class of text-right for right-aligned text.

Use a class of no-layout to kill any padding/margin on elements.

To hide an element on all screens, add a class of hidden.

To hide an element only on smaller screens, a class of hidden-on-mobile does the trick.

In order to only show an element on smaller screens, apply a class of show-only-on-mobile.
