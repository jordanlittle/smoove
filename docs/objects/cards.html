Cards

A very generic class meant to be used for highlighting a certain section with a white background and a subtle box-shadow. Adding a class of card applies the background and shadows. Include a class of card-pad-[size] (using the sizes lg, md, and sm) to add padding of various sizes.

Avoid mixing the grid classes with these card classes. Use the grid to create a scaffold around cards.

Card-pad sizes are used on cards to keep a separate scope from the global pad-[direction] classes.

<div class="u-md-6 u-sm-6">						
	<div class="card  card-pad-md">
		<!-- ... -->
	</div>
</div>
