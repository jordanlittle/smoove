Dropdowns

Dropdowns must be wrapped in a containing element with a class of .has-dropdown. There are two pieces of a dropdown once inside this container: the trigger and the target. The trigger is defined with the data-attribute data-ui-trigger, while the target is defined with data-ui-target. The trigger must also have a data-dropdown attribute which will contain the HTML id of the target element.

<div class="has-dropdown">
    <a href="#" data-dropdown="dd_1" data-ui-trigger>Activate!</a>
    <div class="dropdown" id="dd_1" data-ui-target>
        <!-- ... -->
    </div>
</div>
